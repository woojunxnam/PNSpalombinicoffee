# CLAUDE.md — PNS Coffee 나무 키우기 게임

## 프로젝트 개요
PNS 팔롬비니 브랜드 연계 방치형 웹게임. 커피나무를 키우며 원두 포인트를 수집하는 소프트 픽셀 아트 스타일.
GitHub Pages에서 정적 호스팅. 서버/DB 없음.

라이브 경로: `https://www.pnscoffee.com/game/`
메인 사이트와 독립적으로 동작. 네비게이션 연결은 추후 결정.

---

## 아키텍처: Phaser + DOM 하이브리드

- **Phaser 3 Canvas**: 중앙 나무 영역만 담당 (화분, 나무, 장식, 탭 연출, 파티클)
- **HTML/CSS DOM**: HUD(상단), 액션 바(하단), 오버레이 패널(인벤토리/컬렉션/보상코드)
- 이유: 한국어 텍스트, 폼 입력, 스크롤 리스트는 DOM이 압도적으로 유리

```
┌─ DOM: 상단 HUD (원두 포인트, Water 게이지, 설정) ─┐
├─ Phaser Canvas: 나무 영역 (탭/연출/시각 변화)     ─┤
├─ DOM: 하단 액션 바 (물주기, 가방, 컬렉션)          ─┤
└─ DOM: 오버레이 패널 (슬라이드업, 필요 시)          ─┘
```

## Source of Truth — 게임 수치

### Water 시스템
- 최대: 100, 시작: 70
- 온라인 감소: 20초마다 -1
- 오프라인 감소: 경과 시간 기준, 상한 -35
- 물주기: +20, 쿨다운 12초
- 100 초과 불가
- 0일 때: 클릭 가능, 원두 포인트 획득 가능, 성장 XP 불가, 자동 성장 중지

### 클릭 시스템
- 탭 1회: 원두 포인트 +1 (항상)
- 탭 1회: 성장 XP +0.5 (Water > 0일 때만)
- 연출: squash & stretch, +1 플로팅 텍스트, 잎 반응

### 자동 성장
- Water > 0: 1분당 성장 XP +1
- Water = 0: 중지

### 오프라인 성장
- Water 남아있던 시간만큼만 반영
- 최대 반영 시간: 4시간
- 오프라인 원두 포인트: 없음

### 성장 단계
| Stage | 이름 | XP 범위 | 시각 구현 |
|-------|------|---------|----------|
| 1 | 새싹 | 0–14 | 필수 |
| 2 | 어린 묘목 | 15–49 | 필수 |
| 3 | 작은 커피나무 | 50–119 | 필수 |
| 4 | 꽃 단계 | 120–219 | 완료 (분홍/흰 꽃) |
| 5 | 커피 체리 | 220+ | 완료 (빨간 열매) |

### 수집 해금 조건
- `pot_terracotta_basic`: 시작 지급
- `pot_cafe_brown`: Stage 2 도달
- `pot_drip_pattern`: Stage 3 도달
- `decor_spoon_small`: 누적 물주기 5회
- `decor_bean_sack_mini`: 누적 탭 30회
- `tree_round_leaf_basic`: 시작 지급
- `tree_long_leaf_soft`: Stage 3 구조 준비
