## 철학 (절대 변경 금지)
1. **cozy idle** — 효율/숙제/가챠/배틀 금지
2. **축적은 확정, 랜덤은 가속기** — 스프레드시트 플레이 불가능하게
3. **화면을 바쁘게 만들지 말 것** — 모션 적게, 느리게
4. **브랜드 톤** — 팔롬비니 커피의 따뜻한 이미지

## 완료된 Phase

### Phase 1 — 핵심 루프 (완료)
하이브리드 레이아웃, 나무 Stage 0~2, 탭 연출, Water 시스템, HUD, localStorage, 오프라인 복귀

### Phase 2 — 수집/수확/배경 (완료)
Stage 3~4 (꽃·체리), 수확 시스템, 컬렉션 패널, 장착 시스템, 해금 조건

### Phase 3 — 보상/튜토리얼/폴리시 (완료)
보상 코드(제품→게임), 튜토리얼 4단계, 수확 보너스, 파티클

### Phase 4 — 계정/클라우드 (완료)
Firebase Auth (Google), Firestore 클라우드 세이브, 오프라인→온라인 머지

### Phase A — 온보딩·리텐션 (완료)
로딩 화면, Web Audio 5종, 통계 패널, 일일 출석 보너스

### Phase B/C — PWA·공유·글로벌 (완료)
PWA, 가방, 단계 정보, Web Share, Firestore 글로벌 통계, 재방문 알림, 수확 연출 강화

### Phase D — 팝업 + 카페 창문 + 날씨 (완료, 커밋 `79c420e`)
- `showNotice` 큐 팝업 시스템 (탭하여 닫기)
- 튜토리얼 4장 카드 / 단계 전환 팝업 / 첫 꽃·첫 체리 마일스톤
- 카페 창문 배경 + 실시각 하늘 그라디언트 (9 스톱 lerp)
- 움직이는 구름 + 날씨 시스템 (맑음 60% / 흐림 25% / 비 15%)
- 비 오는 날 Water +10 보너스
- 밤하늘 별 + 해/달/노을

### Phase E — 펫 2종 + 친밀도 (완료, 커밋 `da20558`)
- 크레마(참새): 첫 꽃 시 해금, 창틀에 앉음
- 카푸치노(고양이): 첫 수확 시 해금, 테이블에 앉음
- 친밀도 0~5, 하루 1회 +1
- 펫 idle 애니메이션 (호흡, 깡충, 꼬리 살랑)
- 레벨업 팝업 각 5단계 메시지
- 통계 패널에 친밀도 하트 5개 표시

## 진행 중 / 계획

### Phase F-1 — 시각 폴리시 + 수확 품질 시스템 (다음 세션) ⭐
**우선순위: 최고**

#### 시각 폴리시
- [ ] 나무 스케일 1.4× (potW/potH/stemH/leaf 수치)
- [ ] 펫 스케일 1.7× (_createSparrow, _createCat 내부 수치)
- [ ] 음영 레이어 (옵션 A) — 나무 잎/체리/꽃/펫/화분에 하이라이트+섀도우 1겹씩
  - 옵션 A: Graphics 기반 다층 음영. 리스크 낮고 즉시 폴리시
  - 옵션 C (미래): 픽셀 스프라이트 시트. 게임 성숙 후 6~12개월 뒤 검토

#### 수확 품질 시스템 (내부 복잡 / 외부 감성)
**철학**: 숫자 UI 노출 없음 → 스프레드시트 플레이 불가 / 감성 문장만 노출
- [ ] State 추가:
  - `stage4EnterTime`, `stage4WaterOkSec` — Ripeness 계산용
  - `totalWaterOkSec`, `totalPlaySec` — Care 계산용
  - `coffeeCerts`, `coffeeCertsEarned` — 원두 증서 (미래 카페 쿠폰용)
- [ ] 내부 품질 공식 (7변수, 총 100점):
  - Ripeness 0~30 (Stage 4에서 Water>0 누적시간 / 2시간)
  - Care 0~25 (전체 플레이 Water>30 비율)
  - Weather 0~15 (비15 / 흐림10 / 맑음5)
  - Pet 0~10 (친밀도 합계)
  - Time of day 0~5 (오전 6~9 수확 +5)
  - Streak 0~10 (연속 출석일)
  - Rarity Roll 0~5 (진짜 랜덤)
- [ ] 등급 → 황금원두 지급:
  - 0~34 평범 → 1개 (보장)
  - 35~59 양호 → 2개
  - 60~84 훌륭 → 3개
  - 85~100 특별 → 4개 + **원두 증서 +1**
- [ ] `doHarvest()` 수정 — 품질 계산 + 지급
- [ ] 수확 팝업 리디자인 — 감성 문장 나열 + 등급 별 (숫자 없음)
  - 예: "크레마가 수확을 지켜봤어요 / 비 내린 날의 맛이 배어있어요 / 등급: 특별 수확 ★★★★"
- [ ] Stage 4 체리 색상 Ripeness %에 따라 녹→노→빨강 lerp

#### 원두 증서 기반 심기 (UI는 나중)
- [ ] 통계 패널에 "🏅 원두 증서" 카드 추가 (보유량/누적)
- [ ] 전환 UI/브랜드 연동은 **Phase F-3 이후** — 지금은 데이터 모델만

#### 도감 (Catalog) — 펫 + 기존 아이템
- [ ] 컬렉션 패널에 "🐾 친구들" 그룹 추가
- [ ] 잠긴 펫은 실루엣 + "???" + 수수께끼 힌트
  - 크레마: "첫 꽃이 피는 날에 찾아올 거예요"
  - 카푸치노: "첫 수확의 향기를 따라 올 거예요"
  - 미래 펫 4종도 자리만 미리 만들고 힌트로 티저
- [ ] 기존 pot/decor 잠금 시각 강화 (현재는 🔒, 이제 실루엣 그림자 느낌)

  - Lv 5: 자동 성장 +0.2 XP/분
  - Lv 7: 오프라인 상한 6시간
  - Lv 10: 새 화분 해금
- [ ] 가챠 아님, 순수 확정 축적

### Phase I — 카페 쿠폰 연동 (브랜드 로드맵)

- [ ] 월 전환 한도 (계정당 월 1~2회)
- [ ] 수동 승인 플로우 시작 → 자동화는 뒤에
- [ ] 쿠폰 티어 예시:
  - 3장 → 드립백 1개
  - 8장 → 드립백 세트 / 10% 할인
  - 20장 → 무료 음료
- [ ] 이 페이즈는 브랜드 결정 대기 (코드만 준비)

## 결정 기록 (Decisions Log)

### 2026-04-10
- **수확 품질 공식**: 7변수 내부 계산 / UI에는 숫자 노출 없음, 감성 문장만. 결정 이유: cozy 유저는 랜덤감을 즐김, 공식 노출 시 효율 플레이로 변질 위험.
- **하드 페일 금지**: 모든 수확에 기본 황금원두 1개 보장. 과숙/미숙 페널티 없음 (Ripeness는 단방향 상승).
- **이중 화폐**: 황금원두(인게임) / 원두 증서(실제 카페 쿠폰용) 분리. 증서는 "특별 수확"에서만 드롭.
- **크기 배율**: 나무 1.4×, 펫 1.7×. 현재 캔버스에 여백이 많아서 존재감 부족.
- **도트 품질**: 옵션 A (음영 레이어) 우선, 옵션 C (스프라이트 시트)는 게임 성숙 후.
- **도감 재구성**: 기존 컬렉션에 펫 그룹 추가, 잠긴 항목은 실루엣 + 수수께끼 힌트.
- **편지 세션 뒤로 미룸**: 원래 4단계였으나 펫 2종+기록장 완성 후에 넣어야 내용이 풍부해짐.
